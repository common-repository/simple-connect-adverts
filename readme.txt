== Description ==
Simple Connect & Adverts lets you easily integrate Amazon products into your WordPress pages. By using your own template , you can present the products in the style of your Website. All by using simple shortcodes. First go to the option page `Simple Connect & Adverts Setup`. Here you can set your Amazon Access Key ID, Secret Access Key and Partner ID. Here you can change the HTML/css template of make your own.

Now you can use the following shortcode in your posts:

ItemSearch: [sca type=\"ItemSearch\" title=\"Your own Title\" search=\"Your search tag\" category=\"All\"]
TopSellers [sca type=\"TopSellers\" title=\"Your own Title\" node=\"165793011\"]
NewReleases [sca type=\"NewReleases\" title=\"Your own Title\" node=\"3375251\"]

Some information about the node for TopSellers & NewReleases can be found at
https://docs.aws.amazon.com/de_de/AWSECommerceService/latest/DG/BrowseNodes.html or browsenodes.com

== Installation ==
Just unpack the `simple-connects-adverts` folder into your plugins directory and activate it on your wordpress plugins page.
Then you will have the option `Simple Connect & Adverts` on your wordpress options page.

== Configuration ==

Go to the new option page `Simple Connect & Adverts`. Here you can set your Amazon Access Key ID, Secret Access Key and Partner ID.
